\section{Theorem: N-Way Splay Tree Search Complexity}

\textbf{Statement:} Searching in an N-way splay tree with $n$ nodes takes $O(\log n)$ amortized time.

\section{Proof}

\begin{enumerate}
    \item Find node: $O(\log n)$ worst case (tree height)
    \item Splay to root: $O(\log n)$ amortized (from Theorem 6)
    \item Total: $O(\log n)$ amortized
\end{enumerate}

\textbf{Conclusion:} N-way splay tree search has $O(\log n)$ amortized time complexity.
